9. **deploy-project.png** - 部署项目的示意图

## 图片要求

- 格式：PNG 或 JPG
- 建议尺寸：宽度 800-1200px
- 文件大小：建议小于 500KB
- 清晰度：确保文字和按钮清晰可见
- 语言：建议使用中文界面截图，便于用户理解

## 上传说明

请将对应的图片文件直接上传到此目录下，文件名需要与上述列表完全一致。
